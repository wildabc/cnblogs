枣木夹子

从 Evernote 迁到 Joplin

2023.12.28

在学校时，总是追新各种软件，非升级到最新版本不行，离开学校后，一变为能用即可，不愿折腾。可是种种在线服务，并不总能长久服务，不得不网络搬家，不免令人难受。之前写过，像 Google 这样的大公司，竟也没有恒心，令人对其好感全无。记笔记的软件，自从 Evernote 出来，就一直用着，还买过一年会员来体验。虽然后来多方面为人诟病，尤其是对免费用户的限制越来越多，也没想过改变，也没有迁到引入国内的印象笔记。

可是这个月初，Evernote 严限免费用户，使其相当于只是试用。老用户连新建一则笔记都不行。这下只能找出路了。在网上搜索，发现讨论此事的都不多，可见 Evernote 这个曾经的独角兽，早已是过去式了。在热门的笔记软件中，选用的是 Joplin。倒也没有仔细比较，只是看重两点：一是体积较小，约两百兆，其实也不小了，启动速度也有点慢，只是今日的软件，体积实在惊人；二是 Joplin 本就是以 Evernote 的替代者为口号，对其格式支持较好，包括标签也能支持。

先使用 evernote-backup 将Evernote 的笔记导出，并转为 enex 格式。Joplin 导入时，有 Markdown 和 HTML 两种格式选择。虽然 Evernote 不支持，但我还是存了一些 Markdown 格式的笔记，如果用 Markdown 格式导入，Joplin 会修改特殊字符，反而使得格式混乱。所以我用 HTML 格式导入，如果需要，可以将文本从查看窗口粘贴到编辑窗口，再用 转换插件 将格式变为 Markdown，就可正常渲染了。还有，HTML 格式导入的笔记不能正常显示 PDF，也可手动将其转为 Markdown。

之所以用专门的笔记软件，而不是直接用本地的编辑软件，主要是可以多端同步。Joplin 支持使用 OneDrive 同步，于是我又找回了多年未用的账号。不过软件的移动版在同步时很容易中断，同步很多次才能将全部笔记下载。软件在桌面端就比较简单，且汉化还不完全，移动版可称简陋。而且，软件在我的 iPad 上不能正常编辑，必须使用旧式编辑器（GitHub issue）。总之，用了半个多月，问题不少，只是无奈的迁移。在年底遇上这样的事，更觉可叹。
